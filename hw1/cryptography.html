What is Cryptography ?

Cryptography is the process of hiding or encoding information so that only the intended recipient can read it. The art of cryptography has been used to encode messages for thousands of years and continues to be used today in bank cards, computer passwords, and e-commerce.

https://www.fortinet.com/fr/resources/cyberglossary/what-is-cryptography

There is two types of Cryptography :

Secret Key Cryptography :Secret key cryptography (often called symmetric encryption) is a process where a single shared key is required to both secure the information and subsequently retrieve it. (https://www.sentinelone.com/fr/cybersecurity-101/cybersecurity/what-is-a-secret-key/).

Public Key Cryptography : Public-key (or asymmetric) cryptography relies on a dual-key system. It generates a linked pair : one public, one private, using mathematical one-way functions to ensure that data encrypted by one key can only be decrypted by the other. (https://en.wikipedia.org/wiki/Public-key_cryptography)
