What is Reverse Engineering ?

Reverse engineering is the process of analyzing a system, object, or piece of software to deduce its design, function, and implementation effectively working backward from the finished product to the initial blueprint.


Tools used in Reverse Engineering :

Ghidra : a platform created by the National Security Agency that allows researchers to deconstruct and analyze compiled code. (http://ghidra.net/).

strings command : The strings command is a Linux utility used to extract and display readable text (printable characters) from binary files, such as executable programs, libraries, or object files (https://man7.org/linux/man-pages/man1/strings.1.html)
